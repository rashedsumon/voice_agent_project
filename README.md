# Conversational Voice Agent

## Overview
This project implements a production-ready conversational voice agent that:
- Understands and responds to customer queries in real-time.
- Executes workflows (booking, order tracking, support ticket creation, etc.).
- Escalates to human agents when necessary.
- Provides analytics and monitoring for usage and errors.

## Features
- Real-time speech recognition.
- Natural Language Understanding (NLU) with transformers.
- Task execution and workflow automation.
- Human handoff support.
- Streamlit-based dashboard for interaction.
